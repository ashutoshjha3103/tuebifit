# TueBiFit — Hackathon pitch materials

| File | Use case |
|------|----------|
| **`pitch-deck.marp.md`** | Edit in Cursor/VS Code with the [Marp extension](https://marketplace.visualstudio.com/items?itemName=marp-team.marp-vscode) → export **PDF** or **HTML** |
| **`pitch-deck.html`** | Open in Chrome/Firefox (double-click or `file://`). **Arrow keys** or **click** to change slides. Print → PDF if needed |

## Timing (finals format)

| Block | Time |
|-------|------|
| Pitch | **2 min** — lean on Slides 1–3 + one line from 4–5 |
| Demo | **2 min** — app: plan gen, nutrition, RepCount |
| Q&A | **2 min** |

## Theme

Matches the app: **#0a1929** background, **#00e676** accent, light text **#e3e8ef**.

## Git note

The whole `slides/` folder is in `.gitignore` (e.g. for local `.docx` research). To **version only** the deck, add to `.gitignore`:

```gitignore
slides/
!slides/README.md
!slides/pitch-deck.marp.md
!slides/pitch-deck.html
```

(Adjust if you add more tracked files.)
